# **App Name**: Cinematic Hero

## Core Features:

- Animated Particle Background: Implement a continuously animated particle background using tsparticles. The background consists of floating particles simulating stars or snow. Uses a canvas element managed by react-tsparticles.
- Parallax Mountain Layers: Create multiple layers of mountain silhouettes with a soft parallax effect on scroll or mouse movement.
- Hero Text Entrance: Animate the hero text (name, subtitle, description) to fade and slide in smoothly using Framer Motion.
- Responsive Centering: Ensure all center content (text) is perfectly centered on both mobile and desktop screens, adapting to different screen sizes using TailwindCSS.

## Style Guidelines:

- Primary color: White (#FFFFFF) for the text to ensure strong contrast and readability against the dark background. It conveys purity and modernity, complementing the minimalistic style.
- Background color: Dark black (#0A0A0A) to provide a cinematic and calm atmosphere, making the content stand out.
- Accent color: Soft grey (#B8B8B8) for subtle highlights and to enhance the atmosphere fog effect.
- Font: 'Inter', a grotesque-style sans-serif font suitable for headlines or body text
- Smooth and subtle transitions for text elements using Framer Motion to create a visually appealing and engaging experience.
- Use a centered layout for the main content to maintain focus and balance, ensuring responsiveness across different screen sizes.
- Implement the animated floating particles with varying speeds to enhance the depth and cinematic feel.
